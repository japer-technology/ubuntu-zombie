# Content & Posting Calendar (template)

A two-week launch cadence. Fill in dates and owners. Times are suggestions —
post when your audience is awake (HN/Reddit: weekday mornings US Eastern).

## Pre-launch (T-7 to T-1)

| Day | Channel | Asset | Owner | Status |
| --- | ------- | ----- | ----- | ------ |
| T-7 | Repo | Tidy README, pin `docs/VISION.md`, finalise "About" + topics | | ☐ |
| T-5 | Assets | Produce screenshots + OG banner (see assets manifest) | | ☐ |
| T-3 | Video | Record 60–90s demo (see video/) | | ☐ |
| T-2 | All | Dry-run every command in the copy; verify links | | ☐ |
| T-1 | Press | Send embargoed press kit to relevant writers (optional) | | ☐ |

## Notes
- Don't cross-post the identical wording everywhere; adapt tone per channel.
- HN and Reddit reward authenticity — disclose you're the author, engage
  honestly with criticism, never astroturf.
- Hold one or two feature screenshots back so the week has fresh material.
